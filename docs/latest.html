What changed

Updated skills/osint-journalism/SKILL.md with a new Claim-origin and falsification ladder section.

The upgrade adds a mandatory pre-draft sequence for social/UGC-sensitive claims:

  1. Trace earliest observable origin
  2. Decompose headline into micro-claims (who/what/where/when/impact)
  3. Run targeted counter-evidence searches
  4. Cap confidence when origin remains unresolved
  5. Force headline to match only still-validated micro-claims

Why this improves journalism quality

  • Reduces false certainty from viral repost chains lacking clear origin.
  • Moves contradiction checks earlier, before narrative lock-in.
  • Improves claim-to-evidence transparency and confidence discipline.
