  2. Provisional COVID-19 Death Counts, Rates, and Percent of Total Deaths, by Jurisdiction of Residence
    Why it matters now: provides state/jurisdiction mortality burden context with explicit provisional-lag caveats for rapid risk communication.

  3. CDC WONDER API for Data Query Web Service
    Why it matters now: provides reproducible county/state/national epidemiologic baselines (mortality, natality, cancer, and other domains) to test whether short-run signals are genuinely anomalous.

  4. Crossing Inventory Data (Form 71) - Current
    Why it matters now: enables current crossing-level public safety exposure checks for local routing and emergency planning.

  5. Crossing Inventory Data (Form 71) - Historical
    Why it matters now: allows crossing-risk trend analysis and before/after checks around crossing status and protection changes.

  6. Crossing Inventory Source Data (Form 71) - Current
    Why it matters now: raw-source values support validation and auditability when transformed “friendly” extracts mask field-level nuance.
