What changed

This DATASETS_B cycle adds three new catalog sources:

  1. BLS Public Data API
  2. BEA Data API
  3. ECB Data Portal API / SDW

Why these additions

  • BLS API improves U.S. labor and inflation baseline checks with official time-series endpoints used in employment and price-trajectory claims.
  • BEA API adds first-party U.S. national/regional accounts context (GDP, personal income, trade-linked macro flows) for claim-check denominator discipline.
  • ECB SDW API strengthens euro area monetary and banking corroboration with direct central-bank statistical series.

Together, these additions improve reproducible macroeconomic context across U.S. and euro-area reporting while preserving first-party source quality.
