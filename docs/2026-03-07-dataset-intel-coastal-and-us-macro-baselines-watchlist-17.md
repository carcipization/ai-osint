## What changed

This DATASETS_A cycle adds three new catalog sources:

1. **NOAA CO-OPS Data Retrieval API**
2. **U.S. Census Bureau APIs**
3. **FRED API (St. Louis Fed)**

## Why these additions

- **CO-OPS** improves high-frequency coastal disruption context (water levels, currents, tides, station meteorology).
- **Census APIs** add authoritative U.S. demographic/housing/economic baselines at fine geography levels.
- **FRED API** strengthens macro/financial baseline checks with reproducible time-series access and release metadata.

Together, these fill a practical gap in U.S.-focused structural and operational corroboration layers without overlapping heavily with existing non-U.S. baseline sources.
